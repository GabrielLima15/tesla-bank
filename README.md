## 💼 Tecnologias utilizadas

Para o desenvolvimento deste site utilizei as seguintes tecnologias:

- Next.js;
- TailwindCSS;
- Typescript;
- ScrollReveal;
- AOS Animate;

---

## Executando o projeto

Faça o clone deste projeto em seu computador e depois execute com npm:

```bash
 git clone {url}
 cd tesla-bank
 npm install
 npm run dev
```
